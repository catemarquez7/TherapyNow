# TherapyNow – Aplicación Android

Este proyecto fue desarrollado como parte de la materia **Aplicaciones Móviles**, utilizando **Android Studio**, con el objetivo de crear una aplicación móvil funcional para pacientes de un consultorio terapéutico.

---

## 🎯 Objetivo del proyecto

Desarrollar una aplicación Android aplicando conceptos de desarrollo mobile, incorporando:

- Gestión de usuarios.
- Autenticación mediante Firebase.
- Persistencia de datos.
- Operaciones CRUD.
- Validación de entradas.
- Manejo eficiente de procesos mediante hilos secundarios.

---

## ✨ Características

- Registro de nuevos usuarios.
- Autenticación utilizando Firebase.
- Validación de datos para evitar registros incorrectos o repetidos.
- CRUD de sesiones
- Base de datos utilizando Firebase.
- Interfaz simple y clara.
- Implementación de tareas complejas mediante hilos secundarios para mejorar la experiencia de usuario.

---

## 🧠 Tecnologías utilizadas

- Android Studio
- Java 
- Firebase Authentication
- Firebase Realtime Database / Firestore
- XML (Layouts)
- Android SDK
